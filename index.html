How to Play

  1. Select Level on the top right
  2. Once level is selected, click on Start Game
  3. There will be a number of sections creatd for the level selected
  4. Click on 1 of the sections to take a shot at goal
  5. See if you can score 10/10 goals

By default the game is set to easy

Feel free to end the Game at any time, to restart from the beginning
